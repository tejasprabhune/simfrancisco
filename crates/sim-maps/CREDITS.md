# Credits

## Tile art — LimeZu "Modern Exteriors"

The ground, building, and prop pixel art in this project is derived from the
**Modern Exteriors** asset pack by **LimeZu** (Eduardo Schiavi).

- Pack: Modern Exteriors — https://limezu.itch.io/modernexteriors
- Author: LimeZu — https://limezu.itch.io/
- Native set used: `Modern_Exteriors_32x32/` (32×32 px tiles, used at native resolution)

The generated atlases (`assets/atlas.png`, `assets/facade_atlas.png`) are derivative
works composited and recolored from these tiles for use in this project.
